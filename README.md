# Character Analysis Skill

一个 Claude Code / Claude Agent 的 **Skill**：根据一段或多段文本，深入分析其中的某个人物（真实人物或虚构角色），输出一份**带原文证据支撑**的结构化 Markdown 人物分析报告。

## 特点

- **证据优先，禁止臆造**：每一条性格、动机、行为结论都必须引用原文片段作为证据。
- **区分明示与推断**：直接写明的标为「明示」，需要推理得出的标为「推断」并给出推断链条。
- **忠于文本**：只基于用户提供的文本下结论，不带入外部知识；与常识冲突时以文本为准。
- **多文本整合**：支持跨多段/多个文件整合同一人物的信息，证据标注来源。

## 分析维度

报告涵盖以下维度：

1. 基本信息与关系网
2. 性格特质（附证据表）
3. 动机与目标
4. 行为模式与心理变化弧线
5. 关键事件与转折点
6. 与其他人物的关系
7. 信息不足 / 无法确定

## 安装

在你的项目下新建 `.claude/skills/character-analysis/` 目录，把本仓库的 `SKILL.md` 放进去：

```
你的项目/
└── .claude/
    └── skills/
        └── character-analysis/
            └── SKILL.md
```

> 只有 `SKILL.md` 是 skill 运行所必需的；`README.md` 仅作仓库说明，无需拷贝。

## 使用

在 Claude Code 中调用：

```
/character-analysis 分析下面这段文本里的林冲：<粘贴文本>
```

或先提供文本，再指明要分析的人物。多文件时可分别贴出并标注来源。

## 输出示例

**示例输入文本：**

> 林冲原是东京八十万禁军教头，为人安分守己，凡事忍让。高俅之子高衙内调戏林冲之妻，林冲赶到时举拳要打，看清是上司之子，"先自手软了"。后高俅设计诱他持刀误入白虎堂，将他刺配沧州。一路上公人董超、薛霸百般折磨，欲在野猪林害他性命，幸得鲁智深暗中相救。到沧州后，林冲被派看守草料场。一夜大雪压塌草厅，他外出沽酒避寒，回来时草料场已被陆谦等人放火焚烧。林冲在山神庙外听见陆谦三人谈话，方知一切皆是高俅要置他于死地的毒计。林冲推开庙门，手刃陆谦三人，雪夜投奔梁山。

**示例输出（完整报告）：**

```markdown
# 人物分析：林冲

> 分析基于：用户提供的 1 段文本（《水浒传》林冲相关片段改写）

## 一、基本信息与关系网
- **身份/角色**：东京八十万禁军教头
  > 证据：「林冲原是东京八十万禁军教头」（明示）
- **关系网**：
  - 与**高俅**：上下级 + 仇敌（高俅设计陷害他）
    > 证据：「高俅设计诱他持刀误入白虎堂，将他刺配沧州」（明示）
  - 与**高衙内**：妻子被其调戏的受害者
    > 证据：「高俅之子高衙内调戏林冲之妻」（明示）
  - 与**鲁智深**：受其相救的友人
    > 证据：「幸得鲁智深暗中相救」（明示）
  - 与**陆谦**：被其追杀，最终亲手所杀
    > 证据：「林冲推开庙门，手刃陆谦三人」（明示）

## 二、性格特质
| 特质 | 类型 | 证据 |
|------|------|------|
| 安分守己、隐忍 | 明示 | 「为人安分守己，凡事忍让」 |
| 软弱怕事（前期） | 推断 | 「看清是上司之子，'先自手软了'」——明知妻子受辱仍因对方身份而退缩，推断其畏惧权势 |
| 隐忍的极限 / 刚烈反抗（后期） | 推断 | 「推开庙门，手刃陆谦三人」——长期忍让后彻底爆发，推断其忍耐有底线 |

## 三、动机与目标
- **前期核心动机**：自保、息事宁人，不愿与权贵冲突（推断）
  > 证据：「凡事忍让」「先自手软了」
- **后期核心动机**：复仇与求生——得知必死无疑后选择反击（推断）
  > 证据：「方知一切皆是高俅要置他于死地的毒计」→「手刃陆谦三人」
- **最终目标**：脱离体制、投奔梁山以求活路（明示）
  > 证据：「雪夜投奔梁山」

## 四、行为模式与心理变化弧线
- **典型行为模式**：先忍让退避，被逼到绝境才反抗
  > 证据：从「先自手软了」到「手刃陆谦三人」的对比
- **变化弧线**：安分守己的教头 → 一再隐忍受害 → 真相揭破、忍无可忍 → 杀人落草
  > 证据：「凡事忍让」（起点）→「百般折磨」「放火焚烧」（受迫）→「方知……毒计」（觉醒转折）→「手刃陆谦三人，雪夜投奔梁山」（终点）

## 五、关键事件与转折点
1. **妻子被调戏、举拳又手软**：暴露其畏惧权势、隐忍的性格底色。
   > 证据：「举拳要打，看清是上司之子，'先自手软了'」
2. **误入白虎堂被刺配沧州**：人生由顺转逆的起点。
   > 证据：「诱他持刀误入白虎堂，将他刺配沧州」
3. **山神庙听闻真相**：决定性转折，幻想破灭、彻底觉醒。
   > 证据：「方知一切皆是高俅要置他于死地的毒计」
4. **手刃陆谦、投奔梁山**：性格质变的终点，从忍让者变为反抗者。
   > 证据：「手刃陆谦三人，雪夜投奔梁山」

## 六、与其他人物的关系
- **高俅**：核心对立面，迫害的总主使，是林冲命运转折的根源。
  > 证据：「皆是高俅要置他于死地的毒计」
- **鲁智深**：关键救命恩人，在野猪林化解杀身之祸。
  > 证据：「欲在野猪林害他性命，幸得鲁智深暗中相救」
- **陆谦**：直接执行迫害的爪牙，最终成为林冲反抗的第一个对象。
  > 证据：「林冲在山神庙外听见陆谦三人谈话」「手刃陆谦三人」

## 七、信息不足 / 无法确定
- 文本未提供林冲的年龄、外貌、出身家庭等基本信息。
- 文本未展开其妻子的结局及林冲对家庭的具体情感描写，故未深入分析其情感维度。

## 备注
- 本片段为《水浒传》相关情节的精简改写，分析严格基于所给文本，未引入原著其他章节的信息。
```
